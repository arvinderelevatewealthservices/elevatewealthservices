# ElevateWealthServices Website

## Overview
ElevateWealthServices is a comprehensive financial services platform offering Mutual Fund Advisory, Insurance Solutions, and Trading & Investing services.

## Website Structure

```
ElevateWealthServices/
├── index.html          # Homepage
├── about.html          # About Us page
├── services.html       # Services page
├── contact.html        # Contact Us page
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
├── images/             # Image assets directory
├── CNAME               # Custom domain configuration
└── README.md           # This file
```

Contact and consultation booking:

## How to Deploy

### Option 1: GitHub Pages
1. Push all files to a GitHub repository
2. Ensure CNAME file is at the root level
3. Configure repository settings for GitHub Pages
4. Your site will be live at the domain specified in CNAME

### Option 2: Web Hosting
1. Upload all files to your web hosting provider
2. Update DNS to point to your hosting provider
3. Configure the CNAME record for your domain

### Option 3: Local Testing
1. Open `index.html` in a web browser
2. Navigate through pages using links
3. Test form functionality and responsiveness

## Customization

### Colors
Update the color values in `css/styles.css`:
- Primary color: `#0284c7` (blue)
- Success color: `#16a34a` (green)
- Secondary color: `#a855f7` (purple)

### Content
Edit the HTML files directly:
- Update company information in footer sections
- Modify phone numbers and email addresses
- Change service descriptions
- Update office address and hours

### Images
Place your images in the `images/` directory and reference them:
```html
<img src="images/your-image.jpg" alt="Description">
```
